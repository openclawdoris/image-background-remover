**创建时间:** 2026-03-17

**状态:** 🏗️ MVP 开发中

---

## 🎯 项目简介

AI 驱动的图片处理工具 - 一键移除背景和去水印

## 🛠️ 技术栈

### 前端
- **框架**: Next.js 14 (App Router) + React 18
- **样式**: Tailwind CSS
- **组件**: shadcn/ui 风格
- **图标**: lucide-react

### 后端
- **框架**: FastAPI
- **语言**: Python 3.9+
- **AI 模型**: 
  - 背景移除: rembg (U2Net)
  - 去水印: LaMa (待集成)

### 部署
- **前端**: Vercel / 阿里云 / 腾讯云
- **后端**: Docker + 云服务器
- **存储**: 本地文件 (开发) / OSS (生产)

## 🚀 核心功能

### ✅ 已实现
1. **背景移除** - 使用 rembg (U2Net)
2. **实时预览** - 上传即预览
3. **一键下载** - 处理完直接下载

5. **双工具切换** - 背景移除 / 去水印

## 📝 启动方式

### 一键启动
```bash
cd image-tools-mvp
./start.sh
```

### 分别启动

**后端**:
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

**前端**:
```bash
cd frontend
npm install
npm run dev
```

**访问**: http://localhost:3000
