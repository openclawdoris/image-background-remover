# Image Background Remover

一个极简的图像背景去除工具，验证产品市场匹配。

## 核心价值

- 快速去除图片背景
- 简单直观的用户体验
- 无需注册即可使用

## 技术栈

- **框架**: Next.js 14+ (App Router)
- **UI**: Tailwind CSS + shadcn/ui
- **部署**: Cloudflare Pages
- **API**: remove.bg API

## 开发

```bash
npm install
npm run dev
```

## 部署

部署到 Cloudflare Pages。
